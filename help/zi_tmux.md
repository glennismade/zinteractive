# zi_tmux

## Tmux session manager

Interactive fzf-powered tmux session manager. Lists all sessions with status, window count, and working directory.

### Usage

```bash
zi_tmux    # open session picker (also ctrl-t)
```

### Keybindings

| Key | Action |
|-----|--------|
| enter | attach to session |
| ctrl-n | create new session |
| ctrl-x | kill session (with confirmation) |
